# Proyecto de Innovación Educativa (PIE) - MacroAIComp

Bienvenido/a a la documentación oficial del **Proyecto de Innovación Educativa (PIE): MacroAIComp**, desarrollado en la Universidad de Málaga (UMA).

## ¿Qué es MacroAIComp?

El objetivo principal del proyecto es crear una suite completa de **Cuadernos Interactivos (Jupyter Notebooks)** programados en Python y Julia, que cubran los modelos fundamentales de la **Macroeconomía Dinámica**. 

Nuestra herramienta está diseñada para tender un puente entre la teoría macroeconómica impartida en las aulas universitarias y la experimentación computacional interactiva, fomentando así un enfoque *Learning-by-Doing*.

## Características Principales

*   **Modelos de Tiempo Continuo y Discreto**: Resolutores para ecuaciones diferenciales y ecuaciones en diferencias.
*   **Aproximación Lineal y No Lineal**: Uso de autovalores para análisis de puntos de silla, y algoritmos de disparo (shooting) para dinámicas de transición exactas.
*   **Dualidad de Lenguajes**: Todo está implementado tanto en Python (SciPy/NumPy) como en Julia (DifferentialEquations/Optim), permitiendo comparar rendimiento y legibilidad.
*   **Nivel Pedagógico Alto**: Pensado para alumnos que jamás han tocado un lenguaje de programación.
